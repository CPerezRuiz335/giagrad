class giagrad.mlops.ReLU(*tensors)[source]#

Bases: giagrad.tensor.Context

Abstract class for all operators defined in mathops, reductionsops, etc An operator creates an instance of itself with class method forward that contains the parents of the Tensor created by Tensor.comm()

Operators are just extensions of Tensor class to have Tensor functionality self contained but separated in different files.
